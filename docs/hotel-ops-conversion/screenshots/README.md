# Screenshots

The conversion plan references the files below. To populate them, open
https://hotel-ops.lovable.app/ in Chrome and use the device-toolbar mobile frame
(iPhone 15 Pro or similar) at 1×, then use the OS screenshot shortcut
(`Cmd + Shift + 4` on macOS, drag the mobile frame area).

Save each as PNG into this directory:

| Filename | URL | What it should show |
|---|---|---|
| `01_tickets_dashboard.png` | `/tickets` | Tickets list with KPI strip and grouped sections (Today subtab) |
| `02_ticket_detail.png` | `/tickets/t2` | Ticket detail with Accept & set ETA visible |
| `02b_eta_sheet.png` | `/tickets/t2` after tapping *Accept & set ETA* | ETA bottom sheet open |
| `03_activity_feed.png` | `/activity` | Activity feed, *All* chip selected |
| `04_create_universal.png` | `/create/universal` | Universal request grid + recent rooms |
| `04b_select_room.png` | `/create/universal` after tapping *Find* | Select-room sheet with All rooms |
| `05_create_new_sheet.png` | tap FAB anywhere | Create-new bottom sheet with 3 options |
| `06_filter_dept_sheet.png` | tap filter icon | Filter-by-department sheet |

Note: `02b_eta_sheet`, `04b_select_room`, `05_create_new_sheet`, and
`06_filter_dept_sheet` are inline sheets — capture by triggering the action,
then screenshotting the bottom sheet portion.
