# UI Implementation Rules

## Purpose
Ensures pixel-perfect, responsive UI implementation.

## How to Use
- Follow this when converting Figma → Flutter

## Figma Rules

- Match EXACT spacing
- Match EXACT typography
- Match EXACT colors

## Layout Rules

- Use spacing scale (4, 8, 16, 24)
- Avoid overflow
- Support all devices

## Responsiveness

- Small phone
- Medium phone
- Large phone
- Tablet

## Platform Rules

- Use Cupertino where needed (iOS)
- Maintain consistent UX

## Restrictions

- No hardcoded colors
- No inline styles
- No layout hacks
