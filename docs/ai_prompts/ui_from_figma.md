# Figma to Flutter Prompt

## Input
Figma design description or screenshot

## Requirements

- Pixel-perfect UI
- Exact spacing and alignment
- Use spacing scale (4,8,16,24)
- Responsive for all devices

## Rules

- No hardcoded colors
- Use ThemeManager & TypographyManager
- Extract reusable widgets

## Output

1. Layout breakdown
2. Widget tree
3. Full code
