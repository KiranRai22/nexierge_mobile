# Agent Workflow Rules

## Purpose
Controls how the AI agent should think, act, and generate code.

## How to Use
- Agent must follow this step-by-step workflow

## Workflow

### Step 1: Understand
- Read requirements
- Identify layers (UI, domain, data)

### Step 2: Plan
- Decide providers
- Identify reusable components

### Step 3: Implement
- Follow architecture strictly
- Use base modules

### Step 4: Explain
- Explain decisions
- Mention assumptions

### Step 5: Improve
- Suggest optimizations

## When to Change Code

- Bug fix
- Performance improvement
- Reusability improvement

## Never

- Rewrite working code without reason
